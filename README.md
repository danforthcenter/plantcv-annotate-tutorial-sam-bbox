This tutorial describes how to use PlantCV-Annotate + Napari to collect bounding box data for use with Ultralytics segement anything. There are portions of this tutorial that
are static or commented because they require access to the SAM3 model (accessible via Hugging Face), or because they use an interactive Napari viewer. This tutorial uses bounding boxes for multiple object segmentation. 

## Tutorial tags/keywords

Napari, Bounding Box, Segment Anything, SAM3
